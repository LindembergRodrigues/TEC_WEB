Vamos usar um novo pijama?

Não, não é uma roupa de dormir :)
Trata-se do pijama lançado pela coordenação do curso de computação no início de todo período.

Com base nas disciplinas que já foram cursadas, o sistema irá informar as opções de disciplinas disponíveis para aquele período.
Após informar a matrícula, o sistema retorna as disciplinas recomendadas para aquele período, juntamente com o professor responsável, horário e turma. Caso não tenhamos interesse naquelas disciplinas, podemos optar por informar a quantidade de disciplinas que desejamos cursar ou ainda escolher as disciplinas com base no seu código.

Serão mantidas as regras da UFCG quanto à quantidade de disciplinas/créditos.
